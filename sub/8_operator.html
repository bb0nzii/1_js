Operators(연산자)

- 할당연산자(=) : 변수의 값을 할당함
    ex) let x = 10;

- 덧셈연산자(+) : 값을 더함
    ex) let x = 5;
        let y = 2;
        let z = x + y;

- 곱셈연산자(*) : 값을 곱함
    ex) let q = 5;
    let w = 5;
    let multi = q * w;

- 비교연산자(>) : 값을 비교함
    ex) 변수를 선언한 뒤 값을 비교

js 연산자의 유형

- 산술연산자 : +, -, *, **(거듭제곱), / , %(나눈 나머지), ++(증가0), --(감소)
- 할당연산자
- 비교연산자
- 문자열연산자
- 논리연산자
- 비트연산자
- 삼항연산자

비교연산자 (true & false)

- == : 같음인데 엄격하게 같음은 아님
- === : 타입까지 고려하여 엄격하게 같음
- < , >

Logical Operators (논리연산자)

- &&(and) : 조건이 두 개가 모두 일치해야 true
- ||(or) : 둘 중에 하나라도 맞으면 true
- !(같지않음) : 부정이 진실이고 진실이 거짓임
