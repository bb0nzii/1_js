HTML 이벤트

💖HTML 이벤트는 브라우저가 수행하는 일 or 사용자가 수행하는 일💖

- onchange : HTML요소가 변경될 때
- onclick : 사용자가 HTML 엘리먼트를 클릭할 때
- onmouseover : 사용자가 HTML 요소로 마우스를 이동할 때 (= :hover)
- onmouseout : 사용자가 HTML 요소에서 마우스를 멀리 이동할 때
- onkeydown : 사용자가 키보드에 어떤 키를 누를 때
- onload : 브라우저가 페이지 로딩을 완료할 때

- 페이지 로딩이 완료되었습니다
- 입력 필드가 변경되었습니다
- 버튼이 클릭되었습니다
