Set 집합

- 고유한 값의 컬렉션
- 각 값은 Set에서 한 번만 발생할 수 있음
- 중복(같은 값)을 허용하지 않음.
    ex) BBONZI 아니고 BONZI로 출력됨

Map

- 키와 값을 쌍으로 가지고 있음
- 키 : 모든 데이터의 유형이 됨
- 맵 : 원래의 삽입순서를 기억함
- 여러 개의 값을 가지고 있는데 Map의 값을 가져올 때는 get()이라는 메소드를 사용하여 가져옴
