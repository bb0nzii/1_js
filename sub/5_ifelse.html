Conditional Statements

💖조건문💖
- if : 지정된 조건이 참인 경우 실행할 코드블럭을 지정하는 데 사용       
    syntax
    if(Condition){}
- else : 동일한 조건이 거짓인 경우 실행할 코드블럭을 지정하는 데 사용
- else if : 첫 번째 조건이 거짓인 경우 실행할 코드블럭을 지정하는 데 사용
- switch : 실행할 여러 대체 코드블럭을 지정하는 데 사용

**switch : 조건이 여러 개일 경우 깔끔한 코딩을 위해서 사용      
    switch(expression){
        case x: 
        {}
        break;
    }
